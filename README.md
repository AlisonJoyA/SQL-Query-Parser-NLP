# SQL Query Parser NLP

## Description
SQL Query Parser NLP is a natural language processing tool designed to parse and understand SQL queries written in plain English. This project leverages advanced NLP techniques to convert user-generated text into structured SQL queries, allowing users to easily interact with databases without needing to know SQL syntax.

## Features
- **Natural Language Processing**: Converts plain English queries into SQL statements.
- **Query Validation**: Checks for potential errors and suggests corrections.
- **User-Friendly Interface**: Simplified interaction for users unfamiliar with SQL.
- **Support for Multiple SQL Dialects**: Handles various SQL dialects, including MySQL, PostgreSQL, and SQLite.
- **Extensive Documentation**: Provides detailed guides and examples for users to get started.

## Usage
1. **Installation**: Follow the installation instructions below.
2. **Input a Natural Language Query**: Simply type your question or request in the provided interface.
3. **Receive SQL Query**: The tool will parse the input and return the corresponding SQL query.

## Installation Instructions
To install SQL Query Parser NLP, follow these steps:
1. Clone the repository:
   ```sh
   git clone https://github.com/AlisonJoyA/SQL-Query-Parser-NLP.git
   ```
2. Navigate to the project directory:
   ```sh
   cd SQL-Query-Parser-NLP
   ```
3. Install the required dependencies:
   ```sh
   pip install -r requirements.txt
   ```
4. Run the application:
   ```sh
   python app.py
   ```
